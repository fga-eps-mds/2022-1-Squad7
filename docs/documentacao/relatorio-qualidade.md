# Relatório de resultados dos formulários de validação das Histórias de Usuário (USs)

### Objetivo

<p align = "justify">O objetivo deste documento é apresentar os resultados da aplicação dos formulários de validação das USs concluídas em cada Release liberada pela equipe, respondidos pelos <i>POs</i> do projeto <b>Visualeasy</b>.</p>


## Relato dos resultados da Execução dos Testes de Qualidade em Uso

<p align = "justify">Os formulários de validação das USs são disponibilizados para respostas dos <i>POs</i> logo após as reuniões de liberação de Release.</p>
